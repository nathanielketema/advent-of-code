# Advent of code in zig (2025)

## Usage

1. Clone the repository:

   ```console
   git clone https://github.com/nathanielketema/advent-of-code.git
   ```

2. Run any of the solutions. Example:

   ```console
   zig run src/day_01.zig -- data/day_01.txt
   ```
